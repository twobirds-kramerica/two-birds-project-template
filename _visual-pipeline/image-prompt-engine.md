# Image Prompt Engine
# Two Birds Visual Intelligence Pipeline

---

## Purpose

This guide helps Claude generate precise, high-quality prompts for ImageFX
(Google's AI image generation tool). The goal is professional, trust-building
imagery appropriate for senior-focused digital services.

---

## Prompt Formula

A strong ImageFX prompt has 5 components:

```
[SUBJECT] + [CONTEXT/SETTING] + [EMOTION/TONE] + [STYLE] + [TECHNICAL SPECS]
```

### Example — Hero image for Digital Confidence Centre:
```
Warm close-up portrait of a smiling senior woman in her 70s
sitting at a bright kitchen table with a tablet computer,
natural window light, feeling confident and capable,
photorealistic, warm colour palette, shallow depth of field,
shot on Canon 5D, 85mm lens
```

---

## Style Guide by Project

### Digital Confidence Centre (Brenda)
- Subjects: seniors aged 65-80, diverse, genuine smiles
- Settings: home environments, libraries, community centres
- Tone: warm, capable, empowered — never pitying or condescending
- Avoid: stock-photo stiffness, overly staged poses, fear-based imagery
- Colours: warm neutrals, soft blues, gentle greens

### Kirks Karate
- Subjects: children and adults in gi, focused and disciplined
- Settings: clean dojo, outdoor training, belt ceremonies
- Tone: confident, focused, community-oriented
- Avoid: aggression, fighting imagery, intimidation
- Colours: white, red, black — strong contrast

---

## Prompt Checklist

Before submitting to ImageFX, confirm:
- [ ] Subject is clearly described (age, expression, activity)
- [ ] Setting is specified
- [ ] Emotional tone is named
- [ ] Style direction is included (photorealistic, illustration, etc.)
- [ ] Technical specs if relevant (lighting, lens, angle)
- [ ] Nothing in the prompt violates platform content policies
